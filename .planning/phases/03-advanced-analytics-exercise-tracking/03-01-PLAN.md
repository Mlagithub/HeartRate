3. Detection algorithm (per D-05, D-08):
   - Calculate session average HR
   - If no baseline (less than 7 days data per D-06): return DetectionResult with is_exercise=false, confidence=0.0
   - Threshold: session_avg > resting_avg + 30 BPM (Claude's discretion)
   - Duration check: session must have at least 5 minutes of data
   - Confidence calculation:
     - Base confidence: (session_avg - resting_avg - 20) / 40 (clamped 0.0-1.0)
     - Higher HR = higher confidence
   - Show detection prompt only if confidence >= 0.5 per D-08
